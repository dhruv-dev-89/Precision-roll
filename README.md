# 🎲 Precision-Roll

A fun and interactive dice game built with React.js and Tailwind CSS. Test your luck — pick a number, roll the dice, and score points!

## 🚀 Live Demo
🔗 [https://precision-roll.vercel.app/](https://precision-roll.vercel.app/)

---

## 📌 Features
- 🎲 Interactive dice rolling mechanic
- 🔢 Number selection before each roll
- 📊 Live score tracking
- ❌ Score deduction on wrong guess
- 🔄 Reset score anytime
- 📖 Built-in rules popup
- 📱 Fully responsive design

---

## 🛠️ Tech Stack
- Frontend: React.js, JavaScript (ES6+)
- Styling: Tailwind CSS v4
- Build Tool: Vite
- Deployment: Vercel

---

## 🎮 How to Play
1. Select any number between 1 to 6
2. Click on the dice image to roll
3. If your number matches the dice → you earn that many points
4. If wrong → 2 points will be deducted
5. Click Reset Score to start over

## ⚙️ Installation & Setup
1. Clone the repository
   git clone https://github.com/dhruv-dev-89/Precision-roll.git

2. Navigate to project folder
   cd Precision-roll

4. Install dependencies
   npm install

6. Start development server
   npm run dev
